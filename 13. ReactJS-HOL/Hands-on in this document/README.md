# Blogger App

ReactJS Hands-On Lab demonstrating conditional rendering, lists, keys, and `map()`.

## Components

- Book Details
- Blog Details
- Course Details

## Conditional Rendering Used

The application demonstrates several approaches:

- `if` statements
- ternary operator
- logical `&&`
- conditional element variables / expressions
- optional chaining with conditional output

## List Rendering

Book, blog, and course data are rendered using `map()` and unique React `key` values.

## Sample Output

The application displays three columns:

- Course Details
- Book Details
- Blog Details

The sample data follows the hands-on output, including Master React, Deep Dive into Angular 11, Mongo Essentials, React Learning, and Installation.

## Run

```bash
npm install
npm start
```
