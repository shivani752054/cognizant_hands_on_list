# Cricket App

ReactJS ES6 Hands-On Lab.

Features: ES6 map, arrow functions, filtering, destructuring, spread operator/array merging, and flag-based conditional rendering.

Set `const flag = true` in `App.js` for the player-list view. Set it to `false` for Odd Players, Even Players, and merged Indian Players.

Run with:

```bash
npm install
npm start
```

All files are kept directly at the GitHub repository root; there are no subfolders.
