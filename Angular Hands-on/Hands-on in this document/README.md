# Student Course Portal — Angular v20 Hands-On

Flat GitHub-ready implementation of the **10 Angular hands-on exercises** from the Digital Nurture 5.0 exercise book.

## Covered
1. Angular CLI/project setup and components
2. Data binding, lifecycle hooks, Input/Output
3. Directives and custom pipe
4. Template-driven forms and validation
5. Reactive forms, FormArray and custom/async validators
6. Services and dependency injection
7. Routing, parameters and guards
8. HttpClient, RxJS and interceptors
9. NgRx actions, reducers, effects and selectors
10. Jasmine/TestBed tests

## Run
```bash
npm install
npm run api
```
Open a second terminal:
```bash
npm start
```

Then visit `http://localhost:4200`.

## Tests
```bash
npm test
```

## Important
The exercise book normally expects standard Angular folders. This submission is intentionally **flat** because the repository requirement here is to keep all files at the root with no subfolders. All imports have therefore been adjusted to root-level relative paths.
