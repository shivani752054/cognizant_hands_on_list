# My First React

Hands-on ReactJS lab project.

## Objective

Create a React application named `myfirstreact` and display:

> welcome to the first session of React

as the heading of the page.

## Prerequisites

- Node.js
- npm
- Visual Studio Code

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Project Structure

```text
myfirstreact/

│   ├── App.css
```

## Expected Output

The browser displays the heading:

**welcome to the first session of React**
