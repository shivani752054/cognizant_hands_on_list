# Office Space Rental App

ReactJS JSX Hands-On Lab.

## Requirements implemented

- JSX syntax
- JSX elements and attributes
- Rendering JSX to the DOM
- JavaScript expressions inside JSX
- Office objects containing Name, Rent, and Address
- A list of office objects rendered using `map()`
- Office-space image using the JSX `src` attribute
- Inline conditional CSS for rent:
  - Red when rent is below 60000
  - Green when rent is 60000 or above

## Sample Office

The first office matches the hands-on output:

- Name: DBS
- Rent: Rs. 50000
- Address: Chennai

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000`.
