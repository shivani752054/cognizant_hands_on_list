# Score Calculator App

ReactJS Hands-On Lab: functional components, props, styling, and rendering.

## Task

Create a Student Management Portal application named `scorecalculatorapp`.

The `CalculateScore` functional component accepts:

- Name
- School
- Total
- Goal

It calculates and displays the student's score.

## Sample Data

- Name: Steeve
- School: DNV Public School
- Total: 284 Marks
- Goal: 3
- Score: 94.67%
