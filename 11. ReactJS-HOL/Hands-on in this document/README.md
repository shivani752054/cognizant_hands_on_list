# Event Examples App

ReactJS Hands-On Lab demonstrating React event handling.

## Features

- Counter with Increment and Decrement buttons
- Increment button invokes multiple methods:
  - increments the counter
  - displays `Hello Member!`
- `Say welcome` button passes `welcome` as an argument
- `Click on me` demonstrates a React synthetic click event and displays `I was clicked`
- `CurrencyConvertor` component
- Controlled form inputs
- `handleSubmit` event handling
- Euro-to-Indian-Rupees conversion

The conversion follows the lab screenshot example:

```text
80 Euro = Rs. 6400
```

## Run

```bash
npm install
npm start
```
