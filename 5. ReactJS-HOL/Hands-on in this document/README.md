# Cohorts Tracker

ReactJS Hands-On Lab for styling React components using CSS Modules and inline styles.

## Requirements Implemented

- `CohortDetails.module.css` CSS Module
- `box` CSS class
- Width: `300px`
- Display: `inline-block`
- Margin: `10px`
- Padding: `10px 20px`
- Border: `1px solid black`
- Border radius: `10px`
- `<dt>` font weight: `500`
- CSS Module applied with `className`
- Ongoing cohort headings displayed in green
- Other cohort headings displayed in blue

## Sample Cohorts

The project contains the three cohorts shown in the hands-on expected output:

1. INTADMDF10 - .NET FSD
2. ADM21JF014 - Java FSD
3. CDBJF21025 - Java FSD
